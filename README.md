# Ride Sharing System (C++)

## Description
This project is a console-based Ride Share System developed in C++ using OOP concepts.
It simulates driver dispatching, trip management, rollback functionality, and city routing.

## Features
- Driver and Rider management
- Trip lifecycle (request, assign, cancel)
- Dispatch engine for best driver selection
- Rollback manager for undoing trips
- City graph for distance calculation
- GitHub collaboration with branches and merges

## Technologies
- C++
- Object-Oriented Programming
- Git & GitHub
- VS Code
